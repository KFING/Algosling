Дано число $N$. Необходимо определить, сколько есть бинарных строк длины $N$, в которых ровно $K$ единиц.Формат входных данныхПервая строка входных данных содержит два целых неотрицательных числа $N$ и $K$ $(0 \le K \le N \le 10^6)$.Формат выходных данныхВыведите одно число - ответ на задачу. Так как ответ может быть очень большим, необходимо его вывести по модулю $10^9 + 7$.
